Sample rails application
========================

Requirements
------------

1. Ruby
1. Bundler
1. Postgresql - production deployment
1. Redis
1. ImageMagick

## Getting started

1. Install gems:
   `$ bundle install`
1. Run migrations on database:
   `$ bundle exec rake db:migrate`
1. Start server:
   `$ rails server`
1. Verify it works at http://localost:3000
